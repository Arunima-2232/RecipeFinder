# What's In My Kitchen?
## Discover Recipes Beyond Your Cookbook.

A full-stack recipe assistant that integrates an LLM-powered chatbot with a Retrieval-Augmented Generation (RAG) pipeline to retrieve personalized recipes using semantic search and vector embeddings.

## Features

* Conversational recipe management using an LLM-powered chatbot with CRUD functionality.
* Ingredient-based recipe recommendation
* Save recipe to database
* View selected recipe in detail
* Update existing recipes with variations in ingredients or procedure
* Delete recipes
* Popup notifications for error messages
* Responsive user interface
